buatkan kode analisis sentimen dimana dataset nya dari file csv(bisa buat dummy dulu dengan hanya 1 kolom saja yaitu kolom `text`). analisis sentimen ini menggunakan autolabelling menggunakan llm menggunakan openai openrouter dengan contoh kode berikut

```
from openai import OpenAI
import os

client = OpenAI(
    base_url="https://openrouter.ai/api/v1",
    api_key=os.environ["OPENROUTER_API_KEY"]
)

response = client.chat.completions.create(
    model="openrouter/auto",  # bisa diganti model lain
    messages=[
        {"role": "system", "content": "Kamu adalah asisten AI yang membantu."},
        {"role": "user", "content": "Jelaskan apa itu LLM secara singkat"}
    ]
)

print(response.choices[0].message.content)
```
